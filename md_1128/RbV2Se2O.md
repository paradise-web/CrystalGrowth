> $RbSe + Rb + V₂O₅ + Se + V → RbV₂Se₂O$

## 🌡️ 生长工艺
- **最高/源区温度**: `1200°C`
- **保温时长**: `2h`
- **完整流程**: 
    > RT -> 120°C (3h) -> 120°C (6h) -> 1000°C (8h) -> 1200°C (20h) -> 760°C (72h) -> 650°C (300h) -> RT (随斜降温)

## 📌 备注
原料按化学计量比混合，封入金属管中进行固相反应。无助熔剂或输运剂使用，升温至高温后缓慢降温结晶。最终产物为块状单晶。
