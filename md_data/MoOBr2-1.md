# 🧪 MoO₂Br₂的单晶制备

## ⚖️ 配料表
| 组分 | 质量 (Mass) | 摩尔比 (Ratio) | 备注 (Role) |
| :--- | :--- | :--- | :--- |
| **Mo** | 约0.167g | 1 | Raw Material |
| **MoO₃** | 约0.333g | 2 | Raw Material |
| **C₆Br₆** | 约0.5g | 1 | Transport Agent |

## 🌡️ 生长工艺
- **最高/源区温度**: `700°C`
- **低温区温度**: `600°C`
- **保温时长**: `3天`
- **完整流程**: 
    > RT -> 700°C (升温速率: 2°C/min) -> 600°C (保温3天) -> 随炉自然冷却

## 📌 备注
采用化学气相传输法（CVT），使用C₆Br₆作为输运剂，双温区设计：700°C源区与600°C生长区；石英管密封抽真空至~10⁻² Pa，防止爆裂；目标产物为α-MoCl₃单晶（文中可能笔误，实际应为MoO₂Br₂）
