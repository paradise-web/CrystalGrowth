# 🧪 CVT-3R MoS2
> **📅 日期**:  | **🔥 设备**: 管式炉 | **⚗️ 方法**: CVT

---

## 1. ⚗️ 反应体系 (Reaction)
**方程式**: 
> $Mo + 2S → MoS2
MoS2 + S + MoCl5 (为①的1/10) → 反应继续$

### ⚖️ 配料表
| 组分 (Component) | 质量 (Mass) | 摩尔比 (Ratio) | 备注 (Role) |
| :--- | :--- | :--- | :--- |
| **Mo粉** | 共2g | 1 | 原料 |
| **S粉** | 与Mo粉同量 | 2 | 原料 |
| **MoCl5** | 为①的1/10 | 1+5wt% | 输运剂 |

## 2. 🌡️ 生长工艺 (Process)
- **高温区 (Source)**: `1100℃`
- **低温区 (Sink)**: `900℃`
- **完整流程**: 
    > RT -> 升温至1100℃ -> 保温 -> 降温至900℃ -> 自然冷却至RT

## 4. 📌 备注
实验中观察到黄色透明晶体，但部分管子剩余大量原料；可能由于Cl2过多导致传输速率过快。其中一管密封良好，获得黄色透晶；完整的一管剩余大量原料，继续反应并保温60天。
