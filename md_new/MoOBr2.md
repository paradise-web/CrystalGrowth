# 🧪 MoO₂Br₂的单晶制备
> **📅 日期**: - | **🔥 设备**: Tube Furnace | **⚗️ 方法**: CVT

![原始记录含表征](../img_test/MoOBr2.jpg)


---

## ⚗️ 反应体系
**方程式**: 
> $Mo + 2 MoO₃ + C₆Br₆ → 3 MoO₂Br₂$

## ⚖️ 配料表
| 组分 | 质量 (Mass) | 摩尔比 (Ratio) | 备注 (Role) |
| :--- | :--- | :--- | :--- |
| **Mo** | 0.19 | 1 | Raw Material |
| **MoO₃** | 0.29 | 2 | Raw Material |
| **C₆Br₆** | 0.55 | 1 | Transport Agent |

## 🌡️ 生长工艺
- **最高/源区温度**: `700°C`
- **低温区温度**: `600°C`
- **保温时长**: `2h`
- **完整流程**: 
    > RT -> 700/600°C (10h) -> 700/600°C (2h)

## 🔬 结果表征
| 类型 | 标注 | 描述 |
| :--- | :--- | :--- |
| Photo | **多相混合** | 产物为多相混合物，包含黑色颗粒和红色杂质 |
| Photo | **簇状一维** | 观察到簇状一维晶体生长形态 |
| Microscope | **XRD谱图** | XRD显示存在Mo₄O₁₁峰，表明未完全生成目标产物MoO₂Br₂ |

## 📌 备注
使用C₆Br₆作为输运剂，采用双温区法（700/600°C）进行化学气相传输（CVT）生长；产物中检测到Mo₄O₁₁杂质，说明反应不完全或副反应发生；晶体呈现簇状一维结构，但整体为多相混合。
